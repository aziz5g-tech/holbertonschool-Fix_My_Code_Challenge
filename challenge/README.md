# Fix My Code Challenge

This directory contains solutions to the "Fix My Code Challenge" tasks.

## Tasks

### 0. FizzBuzz (0-fizzbuzz.py)
A Python implementation of FizzBuzz that prints numbers from 1 to n with special rules:
- Print "Fizz" for numbers divisible by 3
- Print "Buzz" for numbers divisible by 5
- Print "FizzBuzz" for numbers divisible by both 3 and 5

**Fixed Issue:** The original code was checking divisibility by 3 or 5 before 15, causing FizzBuzz to print "Fizz" instead of "FizzBuzz".

### 1. Print Square (1-print_square.js)
A JavaScript implementation that prints a square of a given size using "#" characters.

**Fixed Issue:** The original code was hardcoding 4 characters per line instead of using the size parameter.

### 2. Sort (2-sort.rb)
A Ruby implementation that sorts arguments with numbers in descending order followed by strings.

**Fixed Issue:** The original code was sorting numbers in ascending order instead of descending.

### 3. User Password (3-user.py)
A Python User class with password validation functionality.

**Fixed Issue:** The original code had an incorrect password validation implementation.

### 4. Double Linked List (4-delete_dnodeint/)
A C implementation of a doubly linked list with deletion functionality.

**Fixed Issue:** The delete_dnodeint_at_index function had incorrect logic for removing nodes.

## Compilation and Testing

All files end with a newline as required.
